# Respect App PRD

Respect App is a Flutter social media application.

Main features:
- User login and register
- Home feed
- Create posts with text, images, and videos
- Like, comment, reply, repost, and save posts
- User profiles with avatar, cover, bio, and follow button
- Notifications for likes, replies, mentions, and messages
- Search for users and posts
- Communities with posts, members, moderators, reports, and pinned posts
- Reporting system for bad posts
- AI moderation for inappropriate posts and reports
- Private messaging
- Stories
- Admin/moderation tools

Testing goals:
- Find app crashes
- Find UI bugs
- Test login flow
- Test posting flow
- Test media upload
- Test comments and replies
- Test notifications
- Test profile editing
- Test community posting and reports
- Check broken buttons and navigation problems
